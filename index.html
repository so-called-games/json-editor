JSON Editor

This page is modified version of the original JSON Editor interactive demo. You can learn more about JSON Editor on the official GitHub repository.


Output

There you can make changes below and update properties in the editor using "Update" button, copy or download output using "Copy" and "Save" buttons and upload JSON file to the editor using "Open" button.





Validation

This will update whenever the form changes to show validation errors if there are any.

Schema

There you can change the schema and see how the generated form looks. After you make changes, click "Update". To upload schema from file, click "Open", or click "Save" to download the schema.
